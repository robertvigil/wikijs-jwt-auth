About This Example

This is a demonstration of integrating with the JWT authentication service. The page checks for authentication on load, displays user information from the JWT token, and provides logout functionality.

Key Features:

  • Protected page that requires authentication
  • Displays user information from JWT claims
  • Group-based access control ready
  • Shared authentication utilities (auth.js)
  • Responsive design using Pico CSS
